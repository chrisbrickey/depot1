# depot1
This is a Rails app that allows administrators to fill a product catalog and end-users to shop and purchase those items.


## Setup
* Ruby version: 2.4.0p0
* Rails version: 5.0.2
* Start server: `bin/rails server`


## Tests
* Run tests: `bin/rails test`
