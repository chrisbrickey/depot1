# depot1
This Rails app allows administrators to populate a product catalog. Shoppers can view and purchase those items.


## Setup
* Ruby version: 2.4.0p0
* Rails version: 5.0.2
* Start server: `bin/rails server`


## Tests
* Run tests: `bin/rails test`
